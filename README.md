# 💻 Ejercicios TypeScript - Programación Orientada a Objetos

## 📚 Descripción

Este repositorio contiene una colección de ejercicios prácticos desarrollados en **TypeScript** correspondientes a la materia **Programación Orientada a Objetos**.

Los ejercicios fueron realizados como parte de la cursada de **2° año** de la carrera **Programación Full Stack** en la Universidad Provincial de Córdoba.

## 📂 Contenido

En este repositorio se incluyen ejercicios básicos e introductorios sobre:

* 🔹 Variables
* 🔹 Tipos de datos
* 🔹 Operaciones matemáticas
* 🔹 Entrada y salida de datos
* 🔹 Uso de HTML con TypeScript
* 🔹 Lógica de programación
* 🔹 Primeros conceptos aplicados a Programación Orientada a Objetos

* GitHub
